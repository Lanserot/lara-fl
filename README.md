Laravel пет проект для глубокого изучения

# Старт #
- composer install
- npm install
- создаем бд и прописываем в конфиге
- npm run dev (нужно при работе)

# Накатка бд #
- php artisan migrate
- php artisan db:seed

# Перезаписать бд #
- php artisan migrate:refresh
- php artisan db:seed

composer install
npm run dev
